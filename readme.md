* __Frontend__: HTML, CSS, JavaScript, Pug

## Installation
### Environment variables
Database:
```
NODE_ENV=
PORT=
DATABASE=
DATABASE_PASSWORD=
```
JWT configuration:
```
JWT_SECRET=
JWT_EXPIRES_IN=
JWT_COOKIE_EXPIRES_IN=
```
Mailtrap:
```
EMAIL_USERNAME=
EMAIL_PASSWORD=
EMAIL_HOST=
EMAIL_PORT=
EMAIL_FROM=
```
SendInBlue:
```
SENDINBLUE_API=
SENDINBLUE_USERNAME=
SENDINBLUE_PASSWORD=
```
Stripe:
```
STRIPE_SECRET_KEY=
STRIPE_WEBHOOK_SECRET
```
### Install steps
```
$ npm install
create config.env and set your environment variables
$ npm start
$ npm run dev (for development)
$ npm run start:prod (for production)
```
